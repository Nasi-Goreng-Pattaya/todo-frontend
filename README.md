This is a project for assignment of UM WIF2003 Web Programming built using React framework with Vite.

## Prerequisite

- Node.js v16+

## Running the app

Install all the required packages with:

```
npm install --legacy-peer-deps
```

then run the app with:

```
npm run dev
```
